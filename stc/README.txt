1. Зависимости проекта.
================================================================================

    а) Qt5
    б) cmake
    
2. Сборка проекта
================================================================================

$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .

3. Запуск
================================================================================

а) Запуск с использованием тестовой библиотеки, 
   реализующей функцию DoIt()

$ ./calculator ./doit

б) Запуск с использованием явно заданного пути к библиотеке,
   реализующей функцию DoIt()
   
$ ./calculator <path/to/DoIt-lib>

в) Запуск без явного указания библиотеки,
   реализующей функцию DoIt(). Поиск осуществляется по алгоритму,
   описанному в https://doc.qt.io/qt-5/qlibrary.html#fileName-prop

$ ./calculator

4. Краткое руководство пользователя
================================================================================

Интерфейс программы состоит из двух визуальных частей :

а) основная, содержащая элементы управления калькулятора (допускается ввод с клавиатуры);
б) дополнительная, содержащая элементы управления настройками и мониторинга 
   процесса вычисления:
   
   - консоль отображает ход вычисления выражения;
   - таймаут задаёт значение (в секундах) временной задержки для эмуляции 
     длительных вычислений;
   - строка статуса отображает текущие значения размеров очередей.
